## requirments

0. Установлены Node.js & npm  https://nodejs.org/download/
для норм работы указать прокси для npm:
npm config set proxy http://proxy.company.com:8080
npm config set https-proxy http://proxy.company.com:8080

0. Установлены глобально компилятор typescript и сборщик gulp.
Для установки запустить в цмд: npm i -g typescript gulp

## install

Выполнить: npm install

## run

npm start
